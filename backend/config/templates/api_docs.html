🔐 Authentication

/api/auth/
POST /register/
Register a new user account
POST /login/
Authenticate user and receive JWT tokens
POST /logout/ AUTH
Logout and blacklist tokens
POST /refresh/
Refresh access token
GET PATCH /profile/ AUTH
Get or update user profile
POST /change-password/ AUTH
Change user password

🛒 Shopping Cart

/api/cart/ AUTH REQUIRED
GET /
Get user's cart with totals, tax, and delivery charge
POST /add/
Add item to cart (product_id, quantity)
POST /update_item/
Update cart item quantity (item_id, quantity)
POST /remove_item/
Remove item from cart (item_id)
POST /clear/
Clear all cart items

📦 Orders

/api/orders/ AUTH REQUIRED
GET /
List user's orders
GET /{id}/
Get order details
POST /
Create new order (items, delivery info, coupon)

🏷️ Coupons

/api/coupons/
GET /
List available coupons
GET /{id}/
Get coupon details
POST /validate/
Validate coupon code against cart (code, cart_total, cart_items)
POST /apply/ AUTH
Apply coupon to cart (code)
POST /remove/ AUTH
Remove applied coupon from cart
POST /suggested/
Get suggested coupons for cart (cart_total, cart_items)

❤️ Wishlist

/api/wishlist/ AUTH REQUIRED
GET /
Get user's wishlist items
POST /add/
Add product to wishlist (product_id)
POST /remove/
Remove product from wishlist (product_id)
GET /{product_id}/check/
Check if product is in wishlist
POST /clear/
Clear entire wishlist
