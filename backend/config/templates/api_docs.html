🛒 Happy Groceries API

Complete REST API documentation for your grocery shopping needs

Version 1.0

🔐 Authentication

/api/auth/
POST /register/
Register a new user account
POST /login/
Authenticate user and receive JWT tokens
POST /logout/ AUTH
Logout and blacklist tokens
POST /refresh/
Refresh access token
GET PATCH /profile/ AUTH
Get or update user profile
POST /change-password/ AUTH
Change user password

🍎 Products

/api/products/
GET /
List all products with filtering & search
GET /{id}/
Get product details
GET /categories/
List all categories
GET /categories/{id}/
Get category details
GET /featured/
Get featured products
GET /{id}/related/
Get related products

🛒 Shopping Cart

/api/cart/ AUTH REQUIRED
GET /
Get user's cart with totals
POST /add/
Add item to cart
POST /update_item/
Update cart item quantity
DELETE /remove_item/
Remove item from cart
DELETE /clear/
Clear all cart items
GET /count/
Get cart item count

📦 Orders

/api/orders/ AUTH REQUIRED
GET /
List user's orders
GET /{id}/
Get order details
POST /checkout/
Create new order from cart
POST /{id}/cancel/
Cancel an order
GET /active/
Get active orders
GET /past/
Get past orders
GET /statistics/
Get order statistics

🏷️ Coupons

/api/coupons/
GET /
List available coupons
GET /{id}/
Get coupon details
POST /validate/
Validate a coupon code
GET /user_coupons/ AUTH
Get user's available coupons
