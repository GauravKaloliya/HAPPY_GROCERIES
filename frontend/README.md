# Happy Groceries Frontend

React application for the Happy Groceries e-commerce store.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

## Build

```bash
npm run build
```

## Features

- React 18 with hooks
- React Router v6
- Context API for state management
- JWT authentication
- Responsive design
- Dark mode
